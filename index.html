Connection Zone이라는 웹 페이지를 만들게 된 이유

개요

그동안 Front-end 공부를 하면서 HTML, CSS, Javascript에 대한 지식을 정리한 좋은 글들이
파편화된 것 처럼 흩어져 있어서 일일히 북마크를 했어야 했지만
이런 글들로 넘어가는 링크를 한 번에 모아둔 웹 페이지가 있으면 좋을 것 같기에
어떤 식으로 만들까 고민을 해봤다.
처음에는 wiken으로 만들어 보는 것을 생각해봤지만
고민하면서 예전에 만든 'My Web Study Note'라는 토이 프로젝트처럼
내가 직접 만들어보자는 생각에 이러한 기획을 하게되었다.


정리

형식은 예전에 만들었던 My Web study note의 형식을 따를 것이고
프로그래밍을 공부하면서 본 좋은 글들로 넘어가는 링크를 모아놓은
일종의 정류장 같은 느낌으로 제작해보려고 한다...
